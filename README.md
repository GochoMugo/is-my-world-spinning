# is-my-world-spinning

> A Basic Terminal Utility App to check if my favorite services
> and other things are up and running


## usage:

```bash
⇒ is-my-world-spinning

  This is how the world looks like right now!

        ✓ bitbucket
        ✓ github
        ✓ npm

```
